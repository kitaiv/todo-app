# Todo app

This is my small todo app using TypeScript.

Check it out ;3


## Run Locally

Clone the project

```bash
  git clone https://github.com/kitaiv/todo-app.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
